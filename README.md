# Responsive Tailwind Website

Landing page for an art / app design course built with HTML and Tailwind CSS.

## Features
- Pixel-perfect layout based on the Figma mockup
- Fully responsive (mobile, tablet, desktop)
- Tailwind CSS only (no custom CSS except variables)
- Custom animations and smooth hover transitions
- CSS variables for accent colors
